Использование SVG-иконок

Для вставки svg иконок понадобится:

  1. вставка svg-кода link
  2. подключение sass для сдвига background иконки link
  3. Добавить код на html страничку
  4. Изменить атрибут class на нужный
  5. Изменить атрибут xlink:href на нужный с указанием id (xlink:href="#id")
  6. Посмотреть на полный svg link
<svg height="24" width="24" fill="none" fill-rule="evenodd" class="find">
    <use xlink:href="#find"></use>
</svg>
